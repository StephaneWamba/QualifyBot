# Printer Troubleshooting Guide

## Common Printer Issues

### Printer Not Printing

**Symptoms:**
- Printer shows as offline
- Print jobs stuck in queue
- No response from printer

**Solutions:**
1. Check printer power and connection cables
2. Verify printer is online in printer settings
3. Clear print queue and restart print spooler service
4. Check printer drivers are up to date
5. Restart the printer and computer

### Paper Jams

**Symptoms:**
- Paper stuck in printer
- Error message about paper jam
- Printer stops mid-print

**Solutions:**
1. Turn off printer and unplug from power
2. Open all access panels carefully
3. Remove jammed paper gently (pull in direction of paper path)
4. Check for torn paper fragments
5. Clean rollers with lint-free cloth
6. Reload paper correctly and restart printer

### Poor Print Quality

**Symptoms:**
- Faded or streaky prints
- Blurry text or images
- Missing colors

**Solutions:**
1. Run printer's cleaning utility
2. Check ink/toner levels
3. Align print heads
4. Use correct paper type settings
5. Clean print heads manually if needed
6. Replace low ink/toner cartridges

### Network Printer Not Found

**Symptoms:**
- Cannot find printer on network
- Printer shows as offline
- Connection timeout errors

**Solutions:**
1. Verify printer is connected to same network
2. Check printer's IP address hasn't changed
3. Re-add printer using IP address
4. Check firewall settings aren't blocking printer
5. Restart network router if needed
6. Update printer firmware

## Escalation Criteria

Escalate to IT support if:
- Hardware damage is visible
- Printer requires physical repair
- Solutions above don't resolve issue after 2 attempts
- Multiple users affected simultaneously
